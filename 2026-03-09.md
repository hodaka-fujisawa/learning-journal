# 2026年3月10日の学習記録
## 学んだこと
- cd "場所"で移動
- code .でVScodeを開ける
- .md ファイルを開いた状態でCtrl + Shift + Vを押すと書いた内容がどう表示されるか確認できる
- #で見出し
- 見出しの後にスペースを入れること
- #の多さで階層化する
- -で番号なしリスト
- リストの前に空行を入れること
- 数字.で番号付きリスト
- *で囲うと斜体
- **で囲うと太字
- ***で囲うと斜体＋太字
- ~~で囲うと取り消し線
- 文中のインラインコードは'で挟む
- コードブロックは````で挟む
- リンクは[表示テキスト](URL)で表示
- 画像は![代替テキスト](画像のURL)で表示
- 引用は文頭に>を入れる
- 区切り線は---
- [ ]でチェックボックスを表示,[x]でチェックを入れる
